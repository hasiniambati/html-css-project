# Responsive E-Commerce Landing Page

## Description
This project is a fully responsive e-commerce landing page developed using HTML5 and CSS3. 
It focuses on clean UI design, structured layout, and responsiveness across devices without using JavaScript.

## Features
- Responsive navigation bar
- Hero section with call-to-action
- Product cards using CSS Grid
- Features section using Flexbox
- Mobile-friendly layout using media queries

## Technologies Used
- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries

## How to Run
1. Download or clone the repository
2. Open `index.html` in any web browser
